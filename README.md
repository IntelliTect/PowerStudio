PowerStudio
=============
PowerStudio is a Windows PowerShell integration for Visual Studio 2010. We current support
1. Code coloring
2. Built-in and declared variable auto-completion IntelliSense
2. Error detection for tokens and expressions
3. Tooltips for error information
4. Code folding
5. Brace matching
6. Project and file templates

Prerequisites
-------
In order to build and run the code (only needed for contributing to the project), you will need to install a few items.
1. [Visual Studio 2010 SP1](http://www.microsoft.com/downloads/en/details.aspx?FamilyID=75568aa6-8107-475d-948a-ef22627e57a5)
2. [Visual Studio 2010 SP1 SDK](http://www.microsoft.com/downloads/en/details.aspx?FamilyID=21307c23-f0ff-4ef2-a0a4-dca54ddb1e21)
3. [PowerShell 2.0 SDK](http://www.microsoft.com/downloads/en/details.aspx?FamilyID=50633A1F-A665-425D-923C-1A269F8AC084)
